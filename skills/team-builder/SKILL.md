# 编队推荐

## 功能说明

此技能用于帮助博士组建合适的干员编队，根据不同需求推荐最佳干员组合。

当你缺少某些信息时，可以调用联网搜索工具

## 编队类型

### 通用编队
- 常规推图阵容
- 新手开荒阵容
- 高配速刷阵容

### 特殊编队
- 危机合约阵容
- 集成战略阵容
- 生息演算阵容
- 保全派驻阵容

### 功能编队
- 公招tag阵容
- 好友助战阵容
- 芯片本阵容

## 编队原则

### 基础配置（12人）
- 先锋：2名（费用回复）
- 近卫：2-3名（输出/切人）
- 重装：2名（阻挡/保护）
- 狙击：2名（对空/远程）
- 术师：1-2名（法术输出）
- 医疗：2名（治疗）
- 辅助/特种：1-2名（特殊功能）

### 核心概念
- 费用管理
- 阻挡线
- 输出轴
- 技能循环

## 推荐格式

### 编队概览
```
位置1: [干员名] - [作用]
位置2: [干员名] - [作用]
...
```

### 核心干员
- 必带干员及其作用
- 技能选择建议

### 替代方案
- 低星替代
- 同类型替代

## 使用示例

用户可以这样询问：
- "推荐一个开荒阵容"
- "危机合约怎么编队？"
- "没有银灰用什么替代？"
- "肉鸽开局选什么？"
- "新手应该练哪些干员？"

## 回答格式

回答时应包含：
1. 编队整体思路
2. 各位置干员推荐
3. 核心干员说明
4. 替代方案
5. 使用技巧

## 注意事项

- 考虑用户的干员box
- 提供不同练度的方案
- 说明编队的优缺点
- 提供调整建议
